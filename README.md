## ğŸš€ Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites
Make sure you have **Node.js** installed.

### Installation

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/premium-real-estate.git](https://github.com/YOUR_USERNAME/premium-real-estate.git)
    cd premium-real-estate
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run the Project**
    ```bash
    npm start
    # or if using Vite
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) (or the port shown in terminal) to view it in the browser.
